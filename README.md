# 🍔 Sistema de Gestión para Restaurantes Pro

Una solución web automatizada y moderna diseñada para optimizar las operaciones diarias de un restaurante, desde el control de inventario de insumos hasta la gestión de pedidos y reportes financieros.

Este proyecto consolida mis habilidades en el desarrollo backend utilizando **Python y Flask**, junto con un modelado de datos robusto y una arquitectura limpia.

## 🚀 Funcionalidades Clave

* **Gestión de Menú y Productos:** Control total de platillos, categorías, precios e ingredientes.
* **Control de Inventario en Tiempo Real:** Automatización de entradas y salidas de almacén para evitar desabastecimientos.
* **Módulo de Usuarios y Roles:** Autenticación segura para administradores, cajeros y personal de cocina.
* **Panel de Reportes:** Visualización fluida de las ventas diarias y estadísticas de consumo.

## 🛠️ Stack Tecnológico

* **Backend:** Python 3.x & Flask Framework.
* **Base de Datos:** PostgreSQL (Integridad referencial y consultas optimizadas).
* **Frontend:** HTML5 (Jinja2 Templates), CSS3 Estructurado y JavaScript para interactividad.
* **Seguridad:** Uso estricto de variables de entorno (`.env`) para la protección de llaves secretas y credenciales de BD.

## 📁 Estructura del Proyecto

El sistema se rige bajo un estándar modular profesional:
- `app/static`: Hojas de estilo (CSS), scripts interactivos (JS) y recursos multimedia.
- `app/templates`: Vistas dinámicas e interfaces de usuario organizadas por módulos.
- `models.py`: Modelado de la base de datos relacional y lógica de persistencia.
- `routes.py`: Controladores de rutas, endpoints y lógica de negocio principal.
- `config.py`: Parámetros de entorno y configuraciones de seguridad del servidor.
